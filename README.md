# Sleep Health Data Mining Project

This project explores the relationship between lifestyle behaviors and sleep outcomes using the Sleep Health and Lifestyle dataset.

The goal was to identify patterns in behavioral and physiological variables and build a predictive model for sleep quality.
## Team Contributions

This project was completed as part of a Data Mining and Analytics course at Florida State University.

- Joshua Alcantara – Exploratory Data Analysis (EDA) and dataset preparation

Python  
pandas  
NumPy  
scikit-learn

## Dataset

Sleep Health and Lifestyle Dataset from Kaggle.

400 individuals with features including:

- sleep duration
- sleep quality
- stress level
- physical activity
- heart rate
- daily steps
- sleep disorder status

## Project Workflow

1. Data Cleaning and Preparation
2. Exploratory Data Analysis (EDA)
3. Correlation and Behavioral Pattern Analysis
4. Predictive Modeling using Logistic Regression
5. Model Evaluation

## Key Findings

- Stress level had a strong negative correlation with sleep quality.
- Sleep duration and sleep quality were highly correlated.
- Logistic regression model achieved ~97% accuracy in predicting good vs poor sleep.

## Model Evaluation

Accuracy: ~97%

Metrics used:
- Precision
- Recall
- F1 Score
- Confusion Matrix
